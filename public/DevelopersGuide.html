FastProfFact Developer Guide

What is it?

Fast Professor Fact is a webpage for the student of California State University of long Beach to access the information about the profesor, classes, and books they might need during a semester. The information about a class is spread out so it takes too long to find the information. With this tool, student can just click twice and have all the information on one page.

The Latest Version

Details about the latest version can be found at project's Project Information

Features of Webpage

  • Search for professors in CECS program of csulb
  • Find all the classes a professor teaches
  • See what book are used in the class based on the professor and the class.

Components and Dependencies

  • PHP and Html
  • Mysql
  • Stages involving requirements, design, and implementation stage
